Preptastic - Meal Planning Website

Preptastic is a full stack meal planning web application designed to help users organize and manage their meals efficiently. The platform allows users to create and post recipes, browse recipes from others, save favorites, schedule meals by date and time, and generate ingredient lists.


This project was built using HTML, CSS, JavaScript, and PHP. My main contributions focused on developing the recipe creation functionality, implementing the saved recipes feature, and building the recipe browsing page. I also contributed to the design and user experience of these sections to ensure they were both intuitive and visually appealing.

Faculty and Student Information System - Computer Security Projecte

This project was developed as part of a computer security course and focuses on role based access control within a web application. Built primarily using Node.js, the system simulates a dashboard that displays different functionality depending on the user’s access level.


There are two main roles in the system: student and faculty. Students can view and update their personal information, such as their address, while faculty members have extended permissions. Faculty users can modify student records including tuition and GPA, as well as create new student and faculty accounts.
